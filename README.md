## Overview
Comprehensive dataset of key health indicators for Colombia (2020-2025). Includes mental health metrics (depression, anxiety, suicide rates), healthcare workforce data, infectious disease surveillance (COVID-19, dengue, malaria, HIV), non-communicable disease prevalence (hypertension, diabetes, obesity), and healthcare system indicators. Data disaggregated by year and major regions. Sources: DANE,
